О наборе данных
Набор данных по показателям здоровья при сахарном диабете
Обзор
Этот набор данных содержит 100 000 записей о пациентах, предназначенных для прогнозирования риска развития диабета, анализа и применения в приложениях машинного обучения. Набор данных является чистым, предварительно обработанным и готовым к использованию в классификации, регрессии, разработке признаков, статистическом анализе и визуализации данных.

Количество строк: 100 000
Количество столбцов: более 35
Файл: diabetes_dataset.csv
Описание набора данных
Набор данных включает профили пациентов с характеристиками, основанными на демографических данных, образе жизни, семейном анамнезе и клинических измерениях, которые являются достоверными показателями риска развития диабета. Все данные получены с использованием статистических распределений, основанных на реальных медицинских исследованиях, что обеспечивает сохранение конфиденциальности и отражает реалистичные показатели состояния здоровья.

Качество данных
✅ Полнота: нет пропущенных значений или дубликатов
✅ Чистота: все значения находятся в пределах допустимых с медицинской точки зрения диапазонов
✅ Сбалансированные характеристики: Распределение соответствует реалистичным показателям состояния здоровья населения
✅ Целевое распределение: ~20-25% диагностированных случаев (сбалансировано для классификации ОД)
